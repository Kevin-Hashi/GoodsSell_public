removeLast

  • removeLast(): HTMLSpanElement

  • Remove the last of the breadcrumb list from the list. This is where element.removeFromDOM is called.

    Seealso

    Breadcrumb.removeFromDOM()

    Example

    const lastBreadcrumb = breadcrumbList.last as Breadcrumb;
    const removedElement = breadcrumbList.removeLast();
    assert(!breadcrumbList.list.includes(lastBreadcrumb));
    assert(lastBreadcrumb.element === removedElement);

    Returns HTMLSpanElement
